# BHAG Model

## Team Goal: $2B

$2B is the Benson Crew team's sales-volume goal by 2031.

The milestone path comes from the live BHAG builder. When the underlying assumptions change, the path changes with them.

## Community Goal: 10,000 Agents

10,000 agents is the combined ownership-group downline goal at Real Broker, not Benson Crew team headcount.

The downline is already growing in parallel with the team. Benson Crew remains the primary build, and as the team gets stronger and more visible, downline growth should accelerate.

Real Broker is the platform that makes the downline possible and pays revenue share to the ownership group.

## How the Math Works

The long-range model works only if agent capacity and agent productivity improve together.

- The BHAG builder sets the milestone path and required agent counts.
- The Agent Engine shows whether the current business is pacing toward that path.
- If recruiting pace, attrition, productivity, or split assumptions change, the path has to be read again through the live system view.

For the live operating math behind this path, see the [Agent Engine](agent-engine.md).
