# BHAG Model

## Team Goal: $2B

- $2B is the Benson Crew team's sales-volume goal on the 2026-2035 path.
- The live year-by-year target path comes from the BHAG builder.
- When recruiting pace, attrition, productivity, or split assumptions change, the path changes too.

## Community Goal: 10,000 Agents

- 10,000 agents is the combined ownership-group agent-organization goal at Real Broker, not Benson Crew team headcount.
- That agent organization is already growing in parallel with the team.
- Benson Crew remains the main operating business, and as the team gets stronger and more visible, agent-organization growth should accelerate.
- Real Broker is the platform that makes that broader agent organization possible and pays revenue share to the ownership group.

## How to Read This

- This page is the long-range target path for both the team goal and the community goal.
- The BHAG builder sets the year-by-year target path and required agent counts.
- The Agent Engine shows whether the current business is pacing toward that path.
- Actual pace should be read from the operating sources, not inferred from the target path alone.
- Read this page and the Agent Engine together whenever core assumptions move.
- Use the [Agent Engine](agent-engine.md) for the planning terms and model-reading rules behind the path.
