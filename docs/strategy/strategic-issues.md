# Strategic Issues

This document tracks the current structural issues the strategy is designed to solve.

These are not permanent truths. They are the current business problems that require structural attention.

## Current Issues

### 1. Leadership ownership and execution boundaries are not yet fully protected

Execution still risks routing upward instead of surfacing where it belongs.

### 2. Recruiting is understood but not yet fully predictable

The business needs boring, repeatable recruiting rather than bursts of effort.

### 3. Marketing leverage exists but does not always compound

The issue is not activity. The issue is whether output becomes reusable leverage.

### 4. Operations can absorb upstream failures instead of exposing them

That creates fake health and delays structural correction.

### 5. Retention leverage must be made more explicit

The system should clarify what retention actions actually move the business.

### 6. Leadership rescue is still a default failure mode

If the company depends on escalation to Steve to keep moving, the operating system is not doing its job.
