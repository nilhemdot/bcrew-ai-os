# Quarterly Priorities

This document tracks the current quarter’s operating priorities.

## Q1 2026

Q1 2026 is about making the engine operate predictably, not refining strategy.

### Priority 1: Lock Leadership Ownership, KPIs, and Execution Boundaries

Winning looks like:

- one clear owner per vital function
- 3-5 KPIs per function tied to the Agent Engine
- clear handoffs across Marketing, Recruiting, Ops, and Sales
- leaders bring KPI movement and plans, not task updates
- execution problems surface where they occur instead of escalating upward by default

### Priority 2: Build a Predictable Recruiting Engine

Winning looks like:

- recruiting pace is visible weekly against target
- small-team acquisition and individual recruiting consistently create conversations
- MarketMasters builds trust and qualified attention
- Benson Crew converts through proof, standards, and execution
- every yes moves directly into onboarding
- missed pace triggers structural response, not discussion theater

### Priority 3: Make Benson Crew a Compounding Growth Brand

Winning looks like:

- visible proof of execution is published consistently
- agents reuse and benefit from brand assets
- recruiting conversations naturally reference proof
- retention signals remain stable or improve
- brand activity reduces individual effort instead of merely creating more activity
