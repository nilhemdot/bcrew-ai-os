# Core Values

Core values are decision filters, operating standards, and cultural constraints. They are not campaign language or wall art.

## Crew First

The team wins before any individual does.

This value protects culture, collaboration, and mutual responsibility. If a decision benefits one person while weakening the team, it is the wrong decision.

## Raise the Standard

Never settle for the current level of execution.

This value pushes the business to improve systems, coaching, accountability, and outcomes every quarter. "Good enough" is not a strategic position.

## Own the Outcome

If it is yours, deliver it.

This value rejects excuse-making, handoff ambiguity, and passive responsibility. Ownership means the result is carried through, not merely discussed.

## How the Values Are Used

- hiring and firing
- leadership decisions
- team design
- conflict resolution
- standards enforcement

If a strategic choice violates the core values, it should not survive the decision process.

## How to Read This

- This page defines the values as operating filters, not campaign language.
- Read it when a hiring call, leadership decision, standards issue, or conflict needs a clear decision frame.
- If a choice improves short-term output but violates a core value, the value wins.
