This folder is the canonical home for the live rebuild plan.

UI surface: `/foundation#rebuild-plan`

Session records and reconstructed conversation copies live in `docs/handoffs/`.

Read in this order:

1. `owners-closeout.md`
   - exact Owners package closeout order
2. `current-plan.md`
   - current execution order and what is next
3. `current-state.md`
   - tight done / not-done / next view
4. `../rebuild-decisions.md`
   - locked rebuild decisions already made
5. `current-runtime-map.md`
   - plain-English explanation of Foundation, OpenClaw, Harlan, Crewbert, terminal agents, and memory
6. `agent-architecture.md`
   - target shape for Harlan, Crewbert, assistants, delegation, and agent boundaries
7. `rebuild-master-plan.md`
   - archived baseline that keeps the original rebuild intent inside this repo

Old rebuild planning artifacts in the reference repo are background only. This folder is the current in-repo source for rebuild direction.
