# Users

Status: Active
Last reviewed: 2026-04-26

This is the visible, tracked user layer for Foundation.

Use this folder for shared user files that should be visible in the system.

Do **not** use this folder for local-only runtime state like `USER.md`.

## Rules

- `USER.md` stays local-only
- tracked user docs live here
- each real user gets one file
- keep these files operational, not personal
